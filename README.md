# Flowmeter

## Scaling

The completion time for N workflows scales linearly.

| Workflows | Completion Time (Seconds) |
| ------------- | ------------- |
| 10 | 1.142729667 |
| 100 | 5.442777 |
| 250 | 10.36249067 |
| 500 | 18.09005 |
| 1000 | 34.18952967 |
